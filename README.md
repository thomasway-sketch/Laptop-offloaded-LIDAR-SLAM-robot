# Laptop-offloaded-LIDAR-SLAM-robot
Autonomous indoor mapping &amp; navigation robot built with ROS2 — LiDAR SLAM (slam_toolbox) and Nav2, with on-robot sensing offloaded to a laptop over WiFi. With object detection feeding the navigation costmap

Day 1 - decided on architecture, chose jazzy on ubuntu 24.04 becuase it is the current lts supported to 2029 and is native on my existing dual boot. Started on the tutorials on the official jazzy documentation website.

Day 2 - finished the ROS2 beginner CLI tools tutorials.
