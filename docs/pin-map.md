# ESP32 Pin Map - Laptop-Offloaded LiDAR SLAM Robot

| ESP32 pin | Connects to           | Direction (ESP32) | 
|-----------|-----------------------|-------------------|
| GPIO 22   | TB6612 PWMA           | Output            | 
| GPIO 23   | TB6612 PWMB           | Output            | 
| GPIO 21   | TB6612 AIN1           | Output            | 
| GPIO 19   | TB6612 AIN2           | Output            | 
| GPIO 18   | TB6612 BIN1           | Output            | 
| GPIO 4    | TB6612 BIN2           | Output            | 
| GPIO 27   | TB6612 STBY           | Output            | 
| GPIO 34   | Encoder A, left       | Input             | 
| GPIO 35   | Encoder B, left       | Input             | 
| GPIO 36   | Encoder A, right      | Input             | 
| GPIO 39   | Encoder B, right      | Input             | 
| GPIO 16   | RPLIDAR data (UART2 RX) Input             | 
| GPIO 1/3  | — (UART0 console)     | —                 |
